dotfiles

My personal dotfiles for CachyOS, managed with yadm. Tracks shell configs, KDE Plasma setup, terminal, editors, audio processing, and various app settings — everything I use and configure to make my machine feeling like home.

Setup

bashyadm clone https://github.com/laghab/dotfiles.git
yadm bootstrap

Syncing

bashdotfiles-sync

Snapshots installed packages, commits, and pushes. That's it.

What's tracked

Shell and Git config, KDE Plasma layout and shortcuts, Kitty terminal, Fish and Zsh, Fastfetch with custom ASCII logos, EasyEffects audio presets, editor settings (Zed, Micro), Vencord, qBittorrent, systemd user services, and package lists for both pacman and AUR.

System


OS: CachyOS (Arch-based)
DE: KDE Plasma
Shell: Zsh + Fish
Terminal: Kitty
Editor: Zed
